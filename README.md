# The Quiet Canvas

## Overview

A Vite + React + TypeScript project using shadcn-ui and Tailwind CSS.

## Local development

Prerequisites: Node.js (or Bun).

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

If you prefer Bun:

```sh
bun install
bun run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the production bundle and deploy the contents of `dist/` to your preferred hosting provider:

```sh
npm run build
```
